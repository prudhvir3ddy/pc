## Motherboard 

A hub for all components in the system - CPU, RAM, storage, graphic card, distributing them power and enable them to function as unified system 

Components You'll Find

- CPU Socket: Where the processor is installed.
- RAM Slots: For memory modules.
- PCIe Slots: For expansion cards (graphics, sound).
- Storage Connectors: For SSDs and HDDs (SATA, M.2).
- Power Connectors: From the power supply.
- Back Panel I/O: USB ports, audio jacks, Ethernet, display outputs. 

Based on the CPU you picked, the motherboards would get filtered out, different generations are compatible different motherboards

If you purchased an overclocked CPU, you would need a motherboard that supports it. 

Key term: Chipset ( lookout for the features )

Cheaper motherboard doesn't bottleneck your other components, it might not just support overclocking 

Things to look out for: 

- Supported ethernet cable 
- High speed USB-C

## Cooler 

Each CPU requires certain cooling temperature, if we don't keep the cpu at that temperature, we will face thermal throttling which slows down the CPU. 

We need to buy the cooler based on the CPU heat output 

Categories 

- Air cooling 
- Liquid cooling 


## RAM 

Generations: DDR3, DDR4, DDR5, ... 

Buy the RAM based on the latest supported generation by your CPU 

Ensure your motherboard has extra slots for RAM for future upgrades. 

Factors: 

- Generation 
- Speed ( Higher speed is better, ensure the CPU / Motherboard supports it )
- Latency ( CL16 is better than CL18 at the same speed )
- Dual channel ( Using two sticks is better than one - 2x16, 2x8. adds perf boost of ~10% with dual channel ).
- Volate ( ensure motherboard supports this voltage )

## Storage 

- ensure to get an SSD over HDD ( massive difference in speeds. You can get an HDD for storage but operating system and programs should be running on the SSD )

Factors: 

- generation ( latest is better. each generation has cap on speeds )
- read speeds / write speeds 
- you can save money here on buying SSD without heat sink ( ensure your motherboards has the SSD cooling support )
